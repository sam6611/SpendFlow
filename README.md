# 📌 Overview

SpendFlow is a modern AI-powered personal finance platform that enables users to manage expenses, budgets, and lending records using natural language.

Instead of manually entering every expense, users can simply type messages like:

> Paid ₹250 for lunch

or

> Rahul ko ₹500 diye

SpendFlow automatically extracts, categorizes, and stores transactions using Google Gemini AI.

The platform also integrates with Telegram, allowing users to manage their finances directly from chat.
